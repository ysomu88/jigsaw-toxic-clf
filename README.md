# Toxic Comment Classifier
Multi-label text classification using the Jigsaw Toxic Comment dataset.
Built with scikit-learn, MLflow, and FastAPI.
## Phases
- Phase 1: Baseline TF-IDF + Logistic Regression
- Phase 2: MLflow experiment tracking
- Phase 3: FastAPI serving endpoint
- Phase 4: DistilBERT fine-tuning (optional)
